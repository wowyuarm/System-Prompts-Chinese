本项目旨在：

*   **探索与收集：** 收集和整理当前主流或具有代表性的大型语言模型 (LLMs) 和 AI 智能体 (AI Agents) 的系统提示词。我们优先选取那些在社区中具有较高热度、展现出独特能力或具有典型设计思路的AI。
